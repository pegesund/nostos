# Nostos Tutorial

Welcome to the Nostos programming language tutorial! Use the arrow keys to navigate between chapters.

## Navigation

- **Left/Right arrows** or **-/+**: Previous/Next chapter
- **Up/Down arrows**: Scroll within chapter
- **Ctrl+W** or **Esc** or **q**: Close tutorial

## Chapters

1. **Language Basics** - Syntax, expressions, comments, running programs
2. **Functions** - Defining functions, pattern matching, recursion, chaining
3. **Pattern Matching** - Match expressions, destructuring, guards
4. **Lists & Tuples** - Collections, list operations, ranges
5. **Maps & Sets** - Key-value maps, sets, immutable operations
6. **Typed Arrays** - Float32Array, Float64Array, Int64Array for performance
7. **Type System** - Records, variants, generics, type aliases
8. **Traits** - Defining and implementing traits
9. **Builtin Traits** - Show, Eq, Ord, Default, Hash, Serialize
10. **Trait Bounds** - Constraining generic types with traits
11. **Error Handling** - try/catch, throw, panic, Option, Result
12. **Modules & Imports** - Organizing code, use statements
13. **Standard Library** - String, Map, Set, Time, File, Crypto, etc.
14. **JSON** - Parsing, serialization, typed deserialization
15. **Concurrency** - Spawn, message passing, MVars
16. **Async Runtime** - Tokio-based non-blocking I/O
17. **Async I/O & HTTP** - HTTP client/server, parallel requests
18. **Reflection & Eval** - typeInfo, makeRecord, eval
19. **Debugging & Profiling** - REPL commands, :profile, :debug
20. **HTML Templating** - Functional HTML generation
21. **Mutability** - var bindings, persistent data structures
22. **Templates & Metaprogramming** - quote, splice, decorators
